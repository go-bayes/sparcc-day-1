Quick Reference Guides

Causal Diagram Notation

DAG Symbol Guide

  • → Direct causal effect
  • ○ Unobserved/latent variable
  • □ Observed variable
  • ←→ Bidirectional relationship
  • - - Non-causal association

Common Structures

  • Confounder: X ← Z → Y
  • Mediator: X → M → Y
  • Collider: X → C ← Y
  • Instrument: Z → X → Y

EESI Workflow

NoteThe EESI Framework
  1. Estimand: What causal quantity do we want?
  2. Estimator: How will we estimate it?
  3. Estimate: What is our numerical result?
  4. Simulation: How do we validate our approach?

We’ll practice this framework throughout the workshop!

Code Templates

Basic DAG in R

library(dagitty)
library(ggdag)

# create a simple dag
dag <- dagitty('dag {
  Religion -> Cooperation
  SES -> Religion
  SES -> Cooperation
}')

# visualise
ggdag(dag) + theme_dag()

# identify adjustment sets
adjustmentSets(dag, exposure = "Religion", outcome = "Cooperation")

Simulation Template

# simulate confounded relationship
set.seed(123)
n <- 1000

# confounders
ses <- rnorm(n)

# treatment (religion)
religion <- rbinom(n, 1, plogis(0.5 * ses))

# outcome (cooperation)
cooperation <- 2 + 0.5 * religion + 0.3 * ses + rnorm(n)

# naive estimate (biased)
lm(cooperation ~ religion)

# adjusted estimate (unbiased)
lm(cooperation ~ religion + ses)
